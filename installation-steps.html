Installation

Setup

This setup is for VPS/VDS with Windows only! (works also on selfhosted server on your PC)

  • Download -> XAMPP, Visual Studio Code (or any text editor, for editing scripts), Firefox (or any browser with low RAM usage)
  • Open port 30120 (TCP and UTP), there is tutorial how to do it -> showcase (you can skip first 2 steps in that tutorial and just type into searchbar -> Windows Firewall with Advanced Security)
  • Install all downloaded stuff from first step
  • Select these things during XAMPP installation -> showcase
  • Run XAMPP, click on MYSQL, APACHE start buttons so your database will start (XAMPP must be always running with your server)

FiveM Server Installation

  • Download first and second part of server files from your fivem.network account
  • Extract first part, put FXServer folder on desktop
  • Put second part into "resources" folder
  • Open server.cfg and change license key and steam web API key
  • Create database called qbcore
    • Open browser -> localhost/phpmyadmin (or some software that you use for database)
    • Click on NEW on left side -> showcase
  • Import sql into database
    • Click on "qbcore" database
    • Import button
    • Select sql.sql (ignore all red text after importing them)
  • Go to your server folder (FXServer)
    • find server.cfg and open it
    • change license key and steam api key -> showcase
    • if you don't know how to register them, uncle google will helps :]
  • Again go to your server folder
    • Go to FXServer/server and start FXServer.exe (it will open small window )
    • Check this part of video or take a look on official fivem manual
    • Setup txAdmin (after setup it will start booting your server)
    • Join your server via F8 console -> connect IPofYourVPS
    • Have Fun
