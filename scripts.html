Jobs

Radialmenu (F1) has category with job options (not every job has options in F1).
  • Police (qb-policejob, qb-mdt)
    • config.lua
      • WIP
    • Evidence Searching/Collecting
      • If you stand to nearby crime scene (shots fired, murder etc), take flashlight, move around and aim on the ground. You should be able to see blue and red markers.
      • Make sure you got empty evidence bag with you.
    • Commands:
      • /clearblood = Clears nearby blood, evidence bag required.
      • /clearcasings = Clears nearby casings/bullets, evidence bag required.
      • /takedna id = Takes DNA sample from player, evidence bag required.
      • /grantlicense id driver/weapon = Grant a license to somone (example: /granlicense id driver)
      • /revokelicense id driver/weapon = Revoke a license from someone
      • /takedrivinglicense
      • /jail id time = Jail player, time is in minutes
      • /unjail id = Release player from the jail
      • /cuff = Cuff closest player
      • /escort = Escort closest player
      • /sc = Softcuff closest player (so he can walk while cuffed)
      • /depot = Send closest vehicle into depot
      • /impound = Send closest vehicle into police impound
      • /paytow = Pay the towtruck driver
      • /paylawyer = Pay the lawyer
      • /resetarmor
      • /911p msg = Answer /911 calls
      • /mdt = Opens MDT
      • /flagplate plate
      • /unflagplate plate
      • /plateinfo plate
      • /cam id = Check cameras
      • /pobject = Place objects
      • /spikestrip = Place spikes

    Scripts Basic Information

    • Almost every script has configuration file where you can setup almost everything, i recommend to go through all the script files.
    • Admin Commands:
      • /tp id = Teleports on player
      • /tp x y z = Teleports on coords (example /tp 1.0 1.0 1.0)
      • /tpm = Teleports to marker
      • /togglepvp = Toggle PVP on server
      • /addpermission = Give player permission (admin/god)
      • /removepermission = Remove players permission
      • /car name = Spawn vehicle by spawn name (ex. /car adder)
      • /dv = Delete Vehicle
      • /giveitem id name amount = Give item to any player (example: /giveitem 1 weapon_assaultrifle 1)
      • /randomitems
      • /resetinv id = Reset players inventory
      • /givemoney id type(cash/bank/crypto) amount = Give money to player (ex. /givemoney id cash 1000)
      • /setmoney id type(cash/bank/crypto) amount = Set players money (ex. /setmoney id cash 1000)
      • /setjob id job grade = example /setjob 1 police 2
      • /setgang id gang grade = example /setgang 1 ballas 2
      • /clearinv id = Clears player inventory
      • /newdoor = Opens menu to create new doorlocks
      • /kickall = Kick everyone from the server
      • /blips = Toggle player blips
      • /names = Toggle player names
      • /coords
      • /noclip
      • /admincar = Set ownership of vehicle to yourself
      • /announce msg = Announce anything you want
      • /admin = Opens Admin Menu
      • /staffchat msg
      • /warn id reason
      • /checkwarns id
      • /delwarn id warning_number
      • /reporttoggle = Toggle Reports from Players
      • /reportr id = Reply to report
      • /setspeed
      • /setammo

      • /revive id
      • /kill id

    • Traphouses (qb-traphouses)
      • Place where you can sell stolen items or clean markedbills
      • How to get pin code
        • You must be close to location of the traphouse (200 game meters)
        • Take out your gun and try to aim on any NPC around the traphouse
        • If NPC puts his hands up, he should give you note with the pin code.
      • Commands:
        • /entertraphouse
        • /multikeys
